## Headers
Header tags in HTML define the size and style of text elements. Each header is denoted by the letter "h" followed by a number from 1 to 6, with the smallest number representing the largest font size.

| Type | Output |
| ------------- | ------------- |
| ``h1`` | <h1>Header 1</h1>
| ``h2`` | <h2>Header 2</h2>
| ``h3`` | <h3>Header 3</h3>
| ``h4`` | <h4>Header 4</h4>
| ``h5`` | <h5>Header 5</h5>
| ``h6`` | <h6>Header 6</h6>
